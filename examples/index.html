  • Example 1: Bars (with data-based height)

    The height of the bars is based on the barsData. The max height of each bar is defined by $maxHeight(...), while the space between the bars is determined by $yOffset(...). 

      container("c1", _ => {
    _.bars(barsData, [$maxHeight(50), $yOffset(30), $width(150)])
  }, $id("example-1"));

  • Example 2-a: Bars

    In this example, the bars are being horizontally shifted by $xOffset(...). 

      container("c2a", _ => {
    _.bars(barsData, [$maxHeight(50), $yOffset(30), $xOffset(30), $width(150)])
  }, $id("example-2a"));

  • Example 2-b: Bars

    A very basic horizontal bar chart. 

      container("c2b", _ => {
    _.bars(barsData, [$maxWidth(50), $yOffset(30), $height(20)])
  }, $id("example-2b"));

  • Example 2-c: Bars (aligned right)

    A very basic horizontal bar chart. However, now we have aligned it right by using $alignRight(...). 

      container("c2c", _ => {
    _.bars(barsData, [$maxWidth(50), $alignRight(), $yOffset(30), $height(20)])
  }, $id("example-2c"));

  • Example 6: Advanced

      container("c4", _ => {
    _.bars(barsData, [$group("be"), $maxHeight(50), $y(50), $yOffset(30), $width(150)]);
    _.ellipses(ellipsesData, [$group("e1"), $maxDiameter(50), $y(50), $ryOffset(30), $x(150)]);
