Welcome to AncuiJS

A modern JavaScript visualization library that makes it easy to present rich, data-driven visuals.

A fresh Declarative approach to visualizations

Use an expressive list of functions to describe the visual with little to not pre-setup.
let barsData = _a.$data([50, 20, 30, 10, 50], "height");

_a.container("bar-1", _ => {
    _.bars(barsData, [_a.$maxHeight(50), _a.$yOffset(30), _a.$width(150)]);
}, _a.$id("example-bar-1"));

Who does it work?

The data is bound using to a height property using $data([...], "height"). The maximum height of each bar is then controlled by $maxHeight(...), while the space between the bars is determined by $yOffset(...).
